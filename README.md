# Usage (Diamond PL)

### How to run

#### Mac or Linux
```bash
./diamond.sh <diamondfilename>
```

What to write to run the codes:

```shell
python execute.py <diamondfilename>
```

#### Windows
```batch
diamond <diamondfilename>
```
